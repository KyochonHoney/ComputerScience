소프트웨어 종류 : 
1. 응용 소프트웨어 - 컴퓨터 사용자들이 특정 분야의 응용을 목적으로 사용할 수 있도록 개발되는 프로그램( 사용자 프로그램, 응용 패키지 프로그램)

- 워드프로세서
- 스프레드시트
저급언어 : 기계어 ( 0과 1로 이루어진 언어) - 어셈블리어
고급언어 : 

2. 시스템 소프트웨어 - 컴퓨터 하드웨어를 운영하고, 응용 소프트웨어가 동작하기 위한 기반 구조를 형성하는 소프트웨어 ( 인터프리터, 컴파일러, 운영체제) 

-운영체제
역할 :
- CLI -> GUI로 바꿈(Command-Line Interface) - Graphical User Interface
- 컴퓨터의 하드웨어 자원관리 ( 주기억장치 관리 등 )
- 파일 시스템(폴더 관리, 파일 생성, 검색, 백업 및 복구)
- 프로그램 실행 관리

종류
윈도우스
유닉스(UNIX) - 벨 연구소에서 개발됨.
맥OS

-언어 번역기(원시프로그램을 목적 프로그램으로 바꾸어줌)
종류 : 컴파일러(전체), 인터프리터(한문장씩)
